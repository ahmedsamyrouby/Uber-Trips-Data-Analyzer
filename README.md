# Uber Trips Data Analyzer

https://ahmedsamyrouby.github.io/Uber-Trips-Data-Analyzer

## How to Get Your Uber Data

1. Go to https://myprivacy.uber.com/exploreyourdata
2. Sign in with your Uber account
3. Go to the **Download** tab and request a download of your data
4. Once the download is ready, Uber will send you a ZIP file
5. Extract the ZIP file and navigate to: **Uber Data → Rider → `trips_data.csv`**
6. Upload the `trips_data.csv` file to the app
